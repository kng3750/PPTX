# PPTX 생성 웹 애플리케이션

문서 파일과 PPTX 템플릿을 업로드하여 새로운 PPTX 파일을 생성하는 웹 애플리케이션입니다.

## 기능

- 문서 파일 업로드 (PDF, Word, 텍스트 파일 지원)
- PPTX 템플릿 업로드
- 문서 내용을 템플릿에 적용하여 새로운 PPTX 생성
- 생성된 PPTX 파일 다운로드

## 설치 방법

```bash
pip install -r requirements.txt
```

## 실행 방법

```bash
python app.py
```

브라우저에서 `http://localhost:5000` 접속

## 사용 방법

1. 문서 파일 업로드 (PDF, DOCX, TXT)
2. PPTX 템플릿 파일 업로드
3. "PPTX 생성" 버튼 클릭
4. 생성된 PPTX 파일 다운로드
